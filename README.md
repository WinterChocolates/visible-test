## 二、安裝教程
>温聲提示  
>>手机也能装Centos 
>>代码需要安装Git  
>>才能复制代码 

`yum -y install git #安装Git`  

### 1. 获取Root权限  
`sudo su root #权限`   

### 2. 复制 
`cd /         #移步最外层目录`   
`git clone https://gitee.com/ningmengchongshui/Yunzai-Bot-Help.git   #复制  

### 3. 运行  
`sh /Yunzai-Bot-Help/Centos/Centosdemo.sh   #启动`
